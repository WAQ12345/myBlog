🛠️ 工具库项目 ​

QAUtils JavaScript 工具库 ​

项目类型：JavaScript 工具库
技术栈：原生 JavaScript, TypeScript, 零依赖
项目描述：功能完整的 JavaScript 工具库，提供手机号处理、键盘事件监听、设备检测、邮箱验证、文件处理、API 请求等常用功能

核心功能：

  • 手机号验证、格式化、脱敏
  • 键盘事件监听、组合键处理
  • 设备类型检测、屏幕分类
  • 邮箱校验、格式化、脱敏
  • 文件处理、图片压缩
  • API 请求、拦截器、模板
  • URL 验证、解析、提取
  • 本地存储管理
  • 时间日期处理
  • 数据验证（身份证、密码、银行卡）
  • 字符串处理（编码转换、命名格式）

技术亮点：

  • 零配置使用：无需安装任何依赖，直接复制粘贴使用
  • 完全自包含：所有代码都是纯 JavaScript 实现
  • 跨平台兼容：支持所有现代浏览器和 Node.js 环境
  • 多种使用方式：支持 HTML 引入、内联使用、模块导入
  • Mac 系统特殊处理：自动检测 Mac 系统，支持 Command 键等
  • 丰富的配置选项：支持自定义验证规则、格式化选项等

🌐 全栈项目 ​

QA简书 - Nuxt 4.x 全栈博客平台 ​

项目类型：全栈博客平台
技术栈：Nuxt 4.x, Vue 3, TypeScript, Ant Design Vue, MySQL, JWT, 腾讯云 COS
项目描述：基于 Nuxt 4.x 构建的现代化全栈博客平台，支持 SSR、服务端 API、用户认证、文章管理等完整功能

核心功能：

  • 文章管理：Markdown 编辑器、文章发布、分类管理、文集系统
  • 用户系统：用户注册、登录、个人信息管理、头像上传
  • 服务端渲染：SSR 优化 SEO 和首屏性能
  • 文件上传：集成腾讯云 COS，支持图片和文件上传
  • 响应式设计：适配各种设备尺寸

技术亮点：

  • 全栈开发：Nuxt 4.x 统一前后端，无需额外后端服务
  • 类型安全：完整的 TypeScript 类型定义，确保代码质量
  • 自动导入：组件、组合式函数、工具函数自动导入
  • 文件系统路由：基于目录结构自动生成路由
  • MySQL 连接池：高效的数据库连接管理
  • JWT 认证：安全的用户认证机制
  • SSR/SSG：支持服务端渲染和静态站点生成

项目描述：完整的问卷调查系统，包含管理端、用户端和数据模拟服务三个子项目

Animated Portfolio - Next.js 动画作品集 ​

项目类型：动画作品集网站
技术栈：Next.js 15, React 18, Framer Motion, Tailwind CSS, EmailJS
项目描述：使用 Next.js 15 App Router 构建的现代化动画作品集网站，展示了丰富的页面切换动画、滚动交互和响应式设计，专注于展示 Next.js 核心特性和 Framer Motion 动画库的强大能力

核心功能：

  • 流畅页面切换：使用 Framer Motion 实现的页面切换动画，黑色遮罩过渡效果
  • 滚动驱动动画：滚动进度追踪，实现视差滚动和 3D 动画效果
  • 响应式导航：桌面端水平导航 + 移动端汉堡菜单，支持动画过渡
  • 3D 大脑动画：About 页面根据滚动进度旋转的 SVG 动画
  • 作品集展示：Portfolio 页面水平滚动视差效果，展示项目作品
  • 联系表单：集成 EmailJS 的邮件发送功能，支持表单验证
  • 图片优化：Next.js Image 组件 + 自定义图片加载器
  • 静态导出：支持完全静态导出，可部署到任何静态托管服务

技术亮点：

  • Next.js 15 App Router：深入学习最新的 App Router 架构和路由系统
  • 服务端 vs 客户端组件：理解何时使用服务端组件和客户端组件
  • Framer Motion 实践：掌握页面切换、滚动动画、视口触发动画等高级技巧
  • 性能优化：Next.js 的图片优化、字体优化、静态导出等优化策略
  • 响应式设计：Tailwind CSS 响应式设计最佳实践
  • 动画设计模式：学习流畅的页面切换和滚动驱动动画的实现
  • 静态导出：使用 output: "export" 生成完全静态网站，无需服务器
  • 自定义加载器：实现自定义图片加载器，支持外部图片源

  • 年度俯瞰：全年数据统计，热力图可视化
  • 本地存储：所有数据存储在本地，保护隐私
  • 通知提醒：支持每日计划提醒（移动端）
  • 主题切换：支持浅色/深色主题

技术亮点：

  • 跨平台开发：iOS、Android、Web 三端支持
  • 状态管理：Zustand 轻量级状态管理
  • 数据持久化：AsyncStorage 本地存储
  • UI组件：React Native Calendars, React Native Reanimated
  • 数据可视化：进度条、完成率统计、日历视图
