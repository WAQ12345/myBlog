QAUtils JavaScript工具库 ​

项目类型：JavaScript工具库
技术栈：原生JavaScript, TypeScript, 零依赖
项目描述：功能完整的JavaScript工具库，提供手机号处理、键盘事件监听、设备检测、邮箱验证、文件处理、API请求等常用功能

核心功能：

  • 手机号验证、格式化、脱敏
  • 键盘事件监听、组合键处理
  • 设备类型检测、屏幕分类
  • 邮箱校验、格式化、脱敏
  • 文件处理、图片压缩
  • API请求、拦截器、模板
  • URL验证、解析、提取
  • 本地存储管理
  • 时间日期处理
  • 数据验证（身份证、密码、银行卡）
  • 字符串处理（编码转换、命名格式）

技术亮点：

  • 零配置使用：无需安装任何依赖，直接复制粘贴使用
  • 完全自包含：所有代码都是纯JavaScript实现
  • 跨平台兼容：支持所有现代浏览器和Node.js环境
  • 多种使用方式：支持HTML引入、内联使用、模块导入
  • Mac系统特殊处理：自动检测Mac系统，支持Command键等
  • 丰富的配置选项：支持自定义验证规则、格式化选项等

📱 移动端项目 ​

QA问卷星系统 ​

项目类型：全栈问卷调查系统
技术栈：React 18, Next.js 15, TypeScript, Redux Toolkit, Ant Design, Node.js, Mock.js
项目描述：完整的问卷调查系统，包含管理端、用户端和数据模拟服务三个子项目

核心功能：

  • 问卷编辑：可视化问卷编辑器，支持拖拽排序
  • 组件系统：可扩展的问卷组件配置系统
  • 数据统计：实时数据统计和图表展示
  • 用户管理：用户注册、登录、权限管理
  • 问卷填写：用户端问卷填写和提交
  • 数据模拟：Mock服务提供测试数据

技术亮点：

  • 多项目架构：前后端分离，管理端和用户端分离
  • React生态：使用React 18 + Hooks + Redux Toolkit
  • Next.js SSR：用户端使用Next.js实现服务端渲染
  • 拖拽排序：基于@dnd-kit实现组件拖拽排序
  • 状态管理：Redux + Immer + Redux-undo实现复杂状态管理
  • 性能优化：代码分割、懒加载、memo优化
  • 工程化：ESLint + Prettier + Husky + Commitlint
  • 测试体系：Jest + React Testing Library + Storybook
