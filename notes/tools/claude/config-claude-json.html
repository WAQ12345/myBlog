~/.claude.json Config Reference ​

~/.claude.json 各配置项说明；部分可手动编辑，多数为内部状态勿改。MCP 优先级与用法见 mcp-and-proxy。

1. 文件与分类 ​

项目说明
路径~/.claude.json
作用用户级设置、MCP、项目状态、登录与功能缓存等
格式JSON；编辑后保存即生效，部分需重启
类别是否建议手动改
顶层 mcpServers（用户级 MCP）是
安装与统计、提示与功能开关、账号与权限、终端与引导否
projects 下各路径状态仅高级需求

2. mcpServers 结构 ​

  "<name>": {
    "type": "stdio",
    "command": "npx",
    "args": ["-y", "@package/mcp-server", "--api-key", "KEY"],
    "env": {}
  }
}

  • type：stdio / http / sse
  • command：启动命令
  • args：参数数组
  • env：传给 MCP 的环境变量

3. projects（项目级状态） ​

键为项目根目录绝对路径；值含 mcpServers、allowedTools、hasTrustDialogAccepted、exampleFiles、用量统计等。全局 MCP 用顶层 mcpServers；项目专用用 projects.<路径>.mcpServers。

4. 常见操作 ​

  • 添加用户级 MCP：在顶层增加或修改 mcpServers，保存后重启或 /mcp 查看。
  • 清理某项目信任：删除 projects 中对应路径项或重置 hasTrustDialogAccepted 等。
  • 备份：复制 ~/.claude.json 到安全位置；勿提交公开仓库。

敏感信息勿提交；内部字段（Cache、Migration、tengu_ 等）勿随意改。
